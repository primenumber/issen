issen

高速なOthelloAIの実装

ビルドと実行

  • C++14に対応したコンパイラと、 Boost C++ Librariesが必要
  • SSE*とpopcountに対応したCPUでないと動作しません

初期化

$ git submodule init
$ git submodule update
$ bundle install --path=vendor/bundle
$ autoreconf
$ ./configure

コンパイル

$ make

実行

オセロサーバーに接続してプレイする

$ bundle exec ruby server.rb

ffotest

$ ./issen --ffotest < input.pos
