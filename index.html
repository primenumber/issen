issen

高速なOthelloAIの実装

ビルドと実行

  • C++14に対応したコンパイラと、 Boost C++ Librariesが必要
  • AVX2, BMIに対応したCPUでないと動作しません(Haswell以降)

初期化

$ git submodule init
$ git submodule update
$ cmake .

コンパイル

$ make

実行

ffotest

$ ./issen --ffotest < input.pos
